# Voclio Mobile

Flutter mobile app for **Voclio** – organize your day with just your voice.  
This app allows users to create tasks, set reminders, and manage daily schedules using voice or text input.  

## 🚀 Features
- Voice & text task input
- Smart reminders (WhatsApp, Email, Push)
- Daily interactive schedule
- Team collaboration mode
- Cross-platform (iOS & Android)

## 🛠️ Tech Stack
- Flutter (Dart)
- Firebase (Auth, Notifications, Cloud Messaging)
- REST API integration (Voclio Backend)

## 📂 Project Structure
- `lib/` – main source code
- `assets/` – icons, images, fonts
- `test/` – unit & widget tests
